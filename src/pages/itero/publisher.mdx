# Publisher

Itero Team features are available only to Professional and Enterprise users. The user who created the team and pay the bill is the team owner. The team owner can add and remove team members, and can also change the team's subscription plan.

## User Journey

1. Owner or Admin setup the store credentials
2. Owner or Admin verifies build and trigger a publish to the store via the Publisher dashboard
